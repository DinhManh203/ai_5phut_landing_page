Cấu hình YouTube API

Để sử dụng tính năng tự động cập nhật video từ YouTube, bạn cần:

  1. Tạo project trên Google Cloud Console
  2. Kích hoạt YouTube Data API v3
  3. Tạo API Key
  4. Lấy Channel ID của kênh YouTube
  5. Thay thế các giá trị trong code: 
    const YOUTUBE_API_KEY = 'your_api_key_here';
const CHANNEL_ID = 'your_channel_id_here';

Hiện tại website đang sử dụng dữ liệu demo. Cấu hình API để tự động cập nhật video thực tế.
